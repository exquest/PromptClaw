# PromptClaw Memory

## Stable memory rules
Store only:
- stable user preferences
- stable repo preferences
- recurring workflow rules
- enduring decisions worth carrying across sessions

Do not store:
- secrets
- passwords
- tokens or credentials
- transient emotional states
- one-off private details that are not operationally useful
- speculative facts
