2. **`bootstrap_identity` missing from ASGI path (hardening — recurring blocking pattern):** When the systemd unit starts uvicorn targeting `cypherclaw.narrative_api.main:app`, `__main__.py:main()` is never called. Add a FastAPI lifespan event (or module-level call) in `main.py` that calls `bootstrap_identity()` so the ASGI deployment path also mints and persists identity on first boot:

   ```python
   from contextlib import asynccontextmanager

   @asynccontextmanager
   async def _lifespan(app: FastAPI):
       from cypherclaw.first_boot import bootstrap_identity
       bootstrap_identity()
       yield

   app: FastAPI = _build_app()  # pass lifespan=_lifespan to create_app or attach to app
   ```

3. **Integration test for identity persistence (hardening):** Add a test that exercises the startup path (both `__main__` and ASGI) and asserts that the identity file exists after startup, covering both standalone and federated modes.
