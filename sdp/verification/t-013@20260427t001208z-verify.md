## Notes for Lead Agent

1. **Throttle regression**: The old code explicitly passed `last_enqueued_at` to `_should_queue_now` to enforce the 30-minute minimum interval. The new code drops this, consistent with the spec's call signature, but `_generation_last_enqueued_at` is still updated — it's now dead state. 3. **`_generation_clap_centroid` tests**: The helper's three-key lookup + `ear_metrics` nested path + 512-element fallback construction is non-trivial. A short parametrized test would lock in the behavior cheaply.
