## Completeness

**Gap — selector selection not guaranteed (minor):** `range(2)` gives exactly 2 selector calls. In a large library the new generated sample may not rank in the top 2, causing the final journal assertion to fail non-deterministically. The test does not explicitly bias the selector toward the freshly registered sample (e.g. by asserting rank position or boosting recency weight). This could produce a flaky test on a fully populated CypherClaw library.

**Gap — self-listen peak tied to last piece only (minor):** `peak_state` is overwritten each loop iteration; only the final piece's peak survives. If the generated sample is played in piece 0 but peak is silent in piece 1, `peak_state` would still be checked against piece 1. The assertion captures "something made sound" but not specifically "the generated sample made sound."

No such test was added. `bootstrap_identity` is already invoked in `daemon.py:2462`, `cypherclaw_daemon.py:3542`, and `sample_capture_daemon.py:1178`, so the runtime path itself is wired. The missing piece is the regression test.
