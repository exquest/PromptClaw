## Stage 2 — Quote (T-012@20260425T183959Zb)

**Goal:** Trigger the composer to run 3 pieces against a captured descriptor and verify at least one self-quotes the sample (motif/tag match logged).

**Tool:** `composer_quote_verify.py` reuses `capture_known_room_sound()` from Stage 1, drives 3 simulated post-piece `self_quote()` calls against a fake JACK self-bus pre-loaded with the same room tone, then calls `find_motif_tag_match()`.

The self-quote gate in `sample_capture_daemon._should_self_quote()` requires `arc_payoff > 0.6`, `click_count == 0`, `mode != working_ambience`. **Verifier output:** Sonnet 4.6 confirmed PASS WITH NOTES — 5 new tests cover match, no-match, arc-divergence, and CLI exit codes (zero/non-zero match paths). Full suite (3299 tests) green.
