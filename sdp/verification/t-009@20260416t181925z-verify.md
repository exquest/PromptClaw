- legato: 0.95–1.02 ✓
- normal: 0.70–0.85 ✓
- staccato: 0.30–0.50 ✓
- staccatissimo: 0.15–0.25 ✓

Contour adjustments are applied correctly before `dur_sec` is written:
- repeated pitch (`interval == 0`) → `gate_time × 0.92` ✓

Gate time is computed as the midpoint of the style range — deterministic, not randomized within the band. The spec says "per-note gate-time by style" without mandating variation, so this is compliant but lacks per-note differentiation within a style.
