# Task T-003: Implement ModalClient

## Problem Statement
The generation subsystem requires a parallel backend client to support rendering music using Modal (in addition to Replicate). The client must implement the `GenerationClient` Protocol (which currently takes a request and returns a result), compute cost using A10G rates and Modal prediction metadata, and include tests using a mocked Modal backend as well as a gated live test.

## Technical Approach
- Create `my-claw/tools/senseweave/generation/client_modal.py` defining `ModalClient`.
- Implement `ModalClient` with an initialization signature and a `generate(request)` method.
- Assume `GenerationRequest` and `GenerationResult` are duct-typed as `Any` for now to match the existing `client_replicate.py` stub, but return a struct/dataclass matching the expected `GenerationResult` shape or compute the cost internally and return it.
- Use `modal.Function.lookup` (or equivalent Modal SDK primitives) to invoke the function.
- Calculate cost: The A10G per-second rate will be applied to the function's execution time (metadata provided by Modal).
- Create `tests/test_client_modal.py` to mock `modal.Function` and verify the protocol parity with `ReplicateClient` (both having a `generate(request)` method).
- Create `tests/test_generation_live_modal.py` for the `@pytest.mark.live_modal` test.

## Edge Cases
- Modal function raises an exception (e.g., timeout, auth failure): should be caught and propagated cleanly, mirroring `GenerationError` if needed.
- Prediction metadata might lack exact timing if the function fails early; cost should default to 0.0.

## Acceptance Criteria
- `client_modal.py` defines `ModalClient` satisfying `GenerationClient` Protocol (via structural typing/duck-typing matching `ReplicateClient`)
  - **VERIFY:** `python -c "import inspect; from my_claw.tools.senseweave.generation.client_modal import ModalClient; from my_claw.tools.senseweave.generation.client_replicate import ReplicateClient; assert inspect.signature(ModalClient.generate) == inspect.signature(ReplicateClient.generate)"`
- Uses Modal Python SDK to invoke a deployed MusicGen function
  - **VERIFY:** `grep -q "modal.Function.lookup" my-claw/tools/senseweave/generation/client_modal.py`
- Cost computed from Modal prediction metadata × per-second A10G rate
  - **VERIFY:** `grep -q "cost_usd" my-claw/tools/senseweave/generation/client_modal.py`
- Mocked test asserts Protocol parity with `ReplicateClient`
  - **VERIFY:** `pytest tests/test_client_modal.py -v`
- Real Modal call gated behind `@pytest.mark.live_modal` (skipped by default)
  - **VERIFY:** `pytest tests/test_generation_live_modal.py -v -m "not live_modal"`
