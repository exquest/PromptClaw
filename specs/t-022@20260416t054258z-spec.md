# Task T-022 Specification: Build SuperCollider recipe references without coupling tests to live audio

## Problem Statement
The project needs a comprehensive set of SuperCollider recipe references to support curriculum exercises. These recipes must be discoverable from existing curriculum documentation but tested without relying on live audio server execution, which can cause flaky tests and coupling issues.

## Technical Approach
- Create a new directory `my-claw/curriculum/recipes/supercollider/`.
- Create a markdown recipe file for each required topic: MIDI-to-frequency, intervals, scales, chord voicings, swing, polyrhythm, subtractive, FM, additive, granular, physical modeling, spectral freeze, EQ, compression, sidechain, reverb, delay, master bus, groups, buses, patterns, and envelopes.
- Each recipe file will include descriptive text, code examples (SuperCollider language), and required metadata/labels to categorize them.
- Link to these recipes in `my-claw/curriculum/EMSD-102/reference/01-overview.md` and `my-claw/curriculum/EMSD-130/reference/01-overview.md` to ensure they are discoverable.
- Write tests in `tests/test_supercollider_recipes.py` that check for the existence of these files and verify they contain the required frontmatter labels. The tests will parse the markdown directly (static analysis) without booting `scsynth`.

## Edge Cases
- **Missing or malformed frontmatter/labels**: Tests will enforce a strict format.
- **Broken links in curriculum reference docs**: Tests will optionally check that references are properly formatted.

## Acceptance Criteria
- reference recipes exist for MIDI-to-frequency, intervals, scales, chord voicings, swing, polyrhythm, subtractive, FM, additive, granular, physical modeling, spectral freeze, EQ, compression, sidechain, reverb, delay, master bus, groups, buses, patterns, and envelopes.
    - VERIFY: `pytest tests/test_supercollider_recipes.py::test_recipe_files_exist`
- recipes are discoverable from curriculum/reference docs.
    - VERIFY: `pytest tests/test_supercollider_recipes.py::test_recipes_linked_in_curriculum`
- tests verify recipe files and required labels exist.
    - VERIFY: `pytest tests/test_supercollider_recipes.py::test_recipe_files_contain_labels`
