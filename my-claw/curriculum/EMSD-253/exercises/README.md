# EMSD-253 Exercises

## ex01: Genre Strategy Translation

**Objective:** Translate one genre strategy into concrete production choices for rhythm, harmony, synthesis, mix role, and scene-transition behavior.

**Verifier:** `structural`

**Template:**

```json
{
  "genre_strategy": "minimalist_process",
  "reference_traits": [
    "repetition",
    "gradual change",
    "clear pulse"
  ],
  "rhythm_choices": {
    "meter_groove": "pulse",
    "density": "medium"
  },
  "synthesis_palette": [
    "subtractive_pluck",
    "additive_pad"
  ],
  "mix_role": "ensemble",
  "transition_type": "seamless"
}
```

**Expected features:**

- `genre_strategy`
- `reference_traits`
- `rhythm_choices`
- `synthesis_palette`
- `mix_role`
- `transition_type`
