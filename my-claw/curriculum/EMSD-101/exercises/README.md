# EMSD-101 Exercises

## ex01: MIDI-Frequency Converter

**Objective:** Implement bidirectional MIDI/frequency conversion with A440 reference and verify against known pitch values.

**Verifier:** `constraint`

**Template:**

```json
{
  "midi_to_freq": true,
  "freq_to_midi": true,
  "nyquist_correct": true
}
```

**Expected features:**

- `midi_to_freq`
- `freq_to_midi`
- `nyquist_correct`
