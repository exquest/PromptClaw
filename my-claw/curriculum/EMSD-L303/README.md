# EMSD-L303: Project Management for Generative Systems

- Credits: `3`
- Semester: `6`
- Category: `foundations`
- Prerequisites: `None`

Planning and maintaining generative music systems: reliability, monitoring, logging, graceful degradation, and continuous operation.

## Learning Objectives

- Plan system reliability for unattended operation
- Implement monitoring and logging for generative systems
- Design graceful degradation for component failures

## Topics

- System reliability engineering
- Monitoring and alerting strategies
- Logging for generative systems
- Graceful degradation design
- Maintenance scheduling
