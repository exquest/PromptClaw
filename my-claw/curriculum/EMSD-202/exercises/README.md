# EMSD-202 Exercises

## ex01: Sample Catalog Builder

**Objective:** Create a sample metadata catalog entry with required fields for library management and playback.

**Verifier:** `structural`

**Template:**

```json
{
  "sample_id": true,
  "source_path": true,
  "duration_ms": true,
  "sample_rate": true,
  "channels": true,
  "tags": true
}
```

**Expected features:**

- `sample_id`
- `source_path`
- `duration_ms`
- `sample_rate`
- `channels`
- `tags`
