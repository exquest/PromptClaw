# EMSD-304 Exercises

## ex01: Constraint Rule Validator

**Objective:** Write a composition constraint rule set with valid schema covering pitch range, rhythm density, and harmonic bounds.

**Verifier:** `structural`

**Template:**

```json
{
  "rules": true,
  "rule_count": true,
  "pitch_range": true,
  "density_bounds": true,
  "schema_valid": true
}
```

**Expected features:**

- `rules`
- `rule_count`
- `pitch_range`
- `density_bounds`
- `schema_valid`
