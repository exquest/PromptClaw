# EMSD-499 Exercises

## ex01: Arc Metadata Completeness

**Objective:** Produce full arc metadata for a 30-minute composition covering all required production fields across five phases.

**Verifier:** `structural`

**Template:**

```json
{
  "arc_id": true,
  "phase_profiles": true,
  "harmonic_plan": true,
  "rhythm_plan": true,
  "synthesis_palette": true,
  "mix_roles": true,
  "spatial_profile": true
}
```

**Expected features:**

- `arc_id`
- `phase_profiles`
- `harmonic_plan`
- `rhythm_plan`
- `synthesis_palette`
- `mix_roles`
- `spatial_profile`
