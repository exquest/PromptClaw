# EMSD-201 Overview

This reference note anchors `Sound Design for the Electronic Musician` inside CypherClaw.

Patch design across synthesis architectures with emphasis on timbre shaping, filter usage, and envelope design for expressive electronic sound.

## Key Questions

- How does **Subtractive synthesis and filters** improve CypherClaw's runtime behavior?
- How does **FM synthesis and modulator ratios** improve CypherClaw's runtime behavior?
- How does **Additive synthesis and partial control** improve CypherClaw's runtime behavior?
- How does **Filter types, cutoff, and resonance** improve CypherClaw's runtime behavior?
- How does **Envelope design (ADSR, multi-stage)** improve CypherClaw's runtime behavior?
- How does **Modulation routing** improve CypherClaw's runtime behavior?

## Measurable Outcomes

- [ ] Design patches using subtractive, FM, and additive synthesis
- [ ] Shape timbre with filter types and resonance settings
- [ ] Build multi-stage envelopes for amplitude, filter, and pitch
- [ ] Compare synthesis architectures for different timbral goals
