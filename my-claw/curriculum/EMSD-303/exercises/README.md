# EMSD-303 Exercises

## ex01: Arc Density Profiler

**Objective:** Design a 5-phase density profile with monotonically increasing density through Convergence and decreasing into Crystallization.

**Verifier:** `constraint`

**Template:**

```json
{
  "phase_count": true,
  "divination_density": true,
  "convergence_peak": true,
  "crystallization_decay": true,
  "contour_valid": true
}
```

**Expected features:**

- `phase_count`
- `divination_density`
- `convergence_peak`
- `crystallization_decay`
- `contour_valid`
